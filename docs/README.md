# Documentation

This directory contains all project documentation.

## Contents

- **SETUP_GUIDE.md**: Development environment setup and local installation
- **API_DOCUMENTATION.md**: REST API endpoints and specifications
- **ARCHITECTURE_DECISIONS.md**: Architecture Decision Records (ADRs)
- **DEPLOYMENT_GUIDE.md**: Production deployment procedures
- **DEVELOPER_GUIDE.md**: Coding standards and contribution guidelines
- **TROUBLESHOOTING.md**: Common issues and solutions
- **CHANGELOG.md**: Version history and release notes

## Purpose

This directory serves as the single source of truth for project information, ensuring consistency and reducing onboarding time for new team members.

## Maintenance

- Keep documentation synchronized with code changes
- Review documentation during code reviews
- Update examples when APIs change
- Document architectural decisions immediately
